## [1.1.0] - 2021-06-13

Migration to Typescript, no changes in usage for CommonJS (the only supported use).

### Added

- typescript support
- ES module support

### Changed

- CJS support is now a result of the build process


## [1.0.4] - 2020-09-06

### Changed

## [1.0.2] - 2019-08-24
